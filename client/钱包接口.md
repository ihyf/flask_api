# 获取余额
---
URL:{baseurl}/api
## 上行
```json
{
    "jsonrpc": "2.0",
    "id": 1,
    "method": "get_balance",
    "params": {
        "account": "0xe3A81Fc2C6d38763D2C167A602B1b540554f57F4"
    }
}
```
## 下行
```json
{
    "id": 0,
    "jsonrpc": "2.0",
    "result": {
```
